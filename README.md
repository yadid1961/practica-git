# Mi Proyecto de Práctica Git

Este es un proyecto de práctica para aprender a usar **Git** y **Markdown**. 
El objetivo es familiarizarse con el control de versiones y la documentación técnica.

## Instalación

1. Clonar el repositorio
2. Instalar dependencias necesarias
3. Ejecutar el proyecto con el comando correspondiente

## Tecnologías utilizadas

- Git
- Markdown
- Visual Studio Code
- GitHub
